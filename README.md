# welfare-spending-analysis-2025
Este proyecto realiza un análisis estadístico para entender cómo la Secretaría de Bienestar gastó su presupuesto. A través de gráficas y tablas, visualizamos la lógica detrás de la asignación de contratos, identificamos patrones de gasto, la concentración en pocos proveedores y detectamos riesgos mediante el cruces de datos con el SAT.


Las bases de datos analizadas se pueden encontrar en los siguientes enlaces:

* Contratos de Plataforma Integral Compranet 2025: https://comprasmx.buengobierno.gob.mx/datos-abiertos
* Listado Completo Artículo 69-B y Listado Global Artículo 69-Bis: http://omawww.sat.gob.mx/cifras_sat/Paginas/DatosAbiertos/contribuyentes_publicados.html
